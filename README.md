# aespa 小卡代購／切卡下單網站原型

這是一個手機優先的靜態 MVP 原型，用來預覽前台下單與後台管理畫面。

## 預覽方式

在此資料夾啟動靜態伺服器：

```bash
python3 -m http.server 4173
```

然後開啟：

```text
http://localhost:4173
```

## 目前範圍

- 前台：首頁、商品列表、商品詳情、下單表單、完成頁、回填五碼、訂單查詢。
- 後台：總覽、商品管理、庫存、訂單、Email 紀錄、到貨通知。
- 已用假資料模擬 Winter 自留、品項售完、訂單狀態與 Email 狀態。

## 畫面導覽

頁面上方有「前台」與「後台」切換。

前台包含：

- 首頁主視覺
- 商品列表
- 商品詳情
- 品項選擇
- 下單表單
- 回填後五碼
- 訂單查詢

後台包含：

- 總覽卡片
- 商品管理表單
- 庫存管理表
- 訂單列表
- 到貨通知欄位
- Email 發送紀錄

## 假資料設定

- 商品總套數：5
- Winter 自留：2
- Winter 已售：3
- 所以前台會自動讓冬、冬吉、冬寧、套收顯示售完。

## 設計重點

- 背景使用米白。
- 主按鈕使用淡藍灰。
- 次要按鈕使用白底灰框。
- 卡片圓角維持 8px。
- 手機版採單欄閱讀。
- 桌機版商品與後台區塊會改成多欄。
